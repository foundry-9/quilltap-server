# Theme Quick-Switcher

> **[Open this page in Quilltap](/settings?tab=appearance)**

The theme quick-switcher is a convenient sidebar button that gives you instant access to switch between themes and color modes (light/dark/system). It's an optional feature that you can enable or disable based on your preferences.

## What is the Theme Quick-Switcher?

The theme quick-switcher is:

- A **button in the sidebar footer** with a palette icon (paintbrush/colors)
- A **quick menu** for changing themes without opening Settings
- An **instant switcher** that applies theme changes immediately
- An **optional feature** you can enable or disable in Settings

When enabled, it gives you the fastest way to switch themes and color modes on the fly.

## Location

The theme quick-switcher button appears in the **sidebar footer** (bottom left of your screen):

- **Icon** — A palette or paintbrush icon
- **Position** — Next to other footer buttons (Settings, Tools, Quick-Hide)
- **Visibility** — Only appears when enabled in Settings
- **Responsive** — In collapsed sidebar mode, shows icon only

## Enabling the Quick-Switcher

### Step-by-Step

1. Open **Settings** from the sidebar
2. Navigate to the **Appearance** tab
3. Find the option **Show theme selector in navigation**
4. Toggle it to **ON** (enabled)
5. The palette icon appears in the sidebar footer immediately

### Why Enable It?

Enable the quick-switcher if you:

- **Switch themes frequently** — Quick access without opening Settings
- **Adjust color modes** — Toggle light/dark mode instantly
- **Experiment with themes** — Quickly preview different themes
- **Have space** — Your sidebar isn't too crowded with buttons
- **Prefer quick access** — Like having tools immediately available

### Why Disable It?

Disable if you:

- **Rarely switch themes** — Don't need quick access
- **Prefer Settings menu** — Happy changing themes in Settings
- **Want minimal sidebar** — Keep the sidebar uncluttered
- **Use system mode only** — No need for frequent color mode changes

## Disabling the Quick-Switcher

1. Open **Settings**
2. Go to the **Appearance** tab
3. Find **Show theme selector in navigation**
4. Toggle to **OFF** (disabled)
5. The palette icon disappears from the sidebar

You can re-enable it anytime by returning to this setting.

## Using the Quick-Switcher

### Opening the Menu

1. **Look for the palette icon** in the sidebar footer
2. **Click the icon** to open the theme menu
3. A dropdown appears showing available themes and color modes
4. The menu stays open until you select an item or click elsewhere

### Closing the Menu

The menu closes when you:

- **Click a theme** — Menu closes after selection
- **Click a color mode** — Menu closes after selection
- **Click elsewhere** — Clicking outside the menu closes it
- **Press Escape** — Closes the menu without selecting anything

## Selecting a Theme

### Available Themes

The quick-switcher shows:

- **Default theme** — Always available, at the top
- **Bundled themes** — Five themes ship with Quilltap (Art Deco, Earl Grey, Great Estate, Old School, Rains)
- **Installed bundle themes** — Any `.qtap-theme` bundles you've installed
- **Legacy plugin themes** — Any npm-based theme plugins (deprecated)

### Choosing a Theme

1. Open the quick-switcher menu
2. Look at the list of themes
3. See color previews for each theme
4. See the theme name in its own font style (preview)
5. Click any theme to select it
6. The checkmark shows which is currently active
7. Your interface updates instantly

### Previewing Themes

Each theme in the menu shows:

- **Color swatches** — Samples of the theme's primary colors
- **Theme name** — Displayed in the theme's heading font
- **Checkmark** — Shows the currently active theme
- **Dark mode badge** — If the theme supports dark mode

## Selecting Color Modes

### Color Mode Options

Below the list of themes, you'll see three color mode options:

- **Light** — Always use light mode (bright background)
- **Dark** — Always use dark mode (dark background)
- **System** — Follow your computer's preference (recommended)

### Choosing a Color Mode

1. Open the quick-switcher menu
2. Scroll down to see color mode options
3. Each option shows a light/dark/system icon
4. Click any option to switch
5. The checkmark shows the current mode
6. Your interface updates instantly

### What Color Modes Do

- **Light mode** — Bright backgrounds, dark text; use in daytime
- **Dark mode** — Dark backgrounds, light text; use at night
- **System mode** — Automatically matches your OS preference

## Quick-Switcher Appearance

### In Expanded Sidebar

When your sidebar is **not collapsed**:

- **Icon and label** — Shows "Themes" with the palette icon
- **Full visibility** — Easy to find and identify
- **Takes space** — Uses more width than icon-only mode

### In Collapsed Sidebar

When your sidebar **is collapsed**:

- **Icon only** — Shows just the palette icon
- **Compact** — Saves space when you need it
- **Tooltip** — Hover to see "Themes" label
- **Same functionality** — Works exactly the same way

## The Theme Menu Design

### Menu Layout

The quick-switcher menu is organized as:

```
┌─────────────────────┐
│ Default Theme       │ ← Default theme always first
│ ─────────────────── │
│ Great Estate        │ ← Bundled & installed themes
│ Rains               │
│ Earl Grey           │
│ ─────────────────── │
│ ☀ Light             │ ← Color mode options
│ ☀ Dark              │
│ ◐ System            │
└─────────────────────┘
```

### Visual Indicators

- **Checkmark (✓)** — Shows the active theme/color mode
- **Color swatches** — Preview the theme's colors
- **Dividers** — Separate themes from color modes
- **Icons** — Identify color modes (sun for light, moon for dark, etc.)

## Workflow Examples

### Quick Theme Switching for Work

1. You're working on a fantasy project with the Great Estate theme
2. You switch to a sci-fi project (different color mood)
3. Click the quick-switcher palette icon
4. Select the Earl Grey theme
5. Back to fantasy work? Click the palette icon again, select Great Estate
6. No Settings needed, no page refresh, instant switching

### Time-Based Color Mode Switching

1. Morning — Click quick-switcher, ensure Light mode is selected
2. Afternoon — Same theme, stays on Light mode
3. Evening — Click quick-switcher, switch to Dark mode
4. Night — Stay in Dark mode or use System mode
5. Quick adjustments without leaving your work

### Testing Different Themes

1. You want to find your favorite theme
2. Open quick-switcher (palette icon)
3. Click through themes one by one
4. See instant previews of each
5. When you find the one you like, your choice is saved
6. Close Settings, keep working with your new theme

## Tips for Using the Quick-Switcher

1. **Enable it if you switch themes** — Makes theme changes much faster
2. **Disable if you don't use it** — Keep your sidebar clean
3. **Use System mode** — Let your OS preference handle light/dark automatically
4. **Experiment freely** — Switch themes frequently to find your favorite
5. **Combine with other tools** — Use alongside width toggle and quick-hide
6. **Check in Settings too** — More detailed theme information is available there

## Quick-Switcher vs. Settings

### Use the Quick-Switcher When

- You want to **switch themes quickly**
- You're actively **experimenting with themes**
- You're **toggling color modes**
- You want **instant feedback** without opening a menu

### Use Settings When

- You want **detailed theme information**
- You're **reading theme descriptions**
- You want **theme history** or comparison
- You prefer a **full Settings interface**

## Troubleshooting

### Quick-Switcher Button Doesn't Appear

**Problem:** The palette icon isn't in the sidebar footer

**Solutions:**

- Check Settings → Appearance → "Show theme selector in navigation"
- Make sure it's enabled (toggle is ON)
- Refresh the page if needed
- Check that you're not in a collapsed sidebar mode where it's hidden

### Menu Won't Close

**Problem:** The theme menu stays open

**Solutions:**

- Click a theme or color mode to make a selection
- Click elsewhere on the page
- Press Escape key
- If stuck, refresh the page

### Themes Don't Change

**Problem:** Clicking a theme doesn't switch it

**Solutions:**

- Check your internet connection
- Verify you're logged in
- Try clicking again
- Refresh the page
- Check browser console for errors

### Wrong Colors Display

**Problem:** The theme colors look incorrect

**Solutions:**

- Check your color mode (Light/Dark/System)
- Ensure the correct theme is selected
- Try switching to a different theme
- Verify your system's light/dark preference if using System mode

### Quick-Switcher is Too Crowded

**Problem:** Too many buttons in the sidebar footer

**Solutions:**

- Disable the quick-switcher if you don't use it frequently
- Collapse your sidebar to icon-only mode
- Disable other optional buttons (Quick-Hide) if not needed
- Use Settings directly instead of quick-switcher

## Advanced: Keyboard Navigation

Currently, the theme quick-switcher is accessed by clicking the icon. Future versions may include:

- Keyboard shortcuts for switching themes
- Numbered theme selection
- Quick color mode toggle hotkeys

Check the Themes help file and Settings for any updates.

## Your Preferences Are Saved

Your quick-switcher settings are automatically saved:

- **Which theme you prefer** — Your theme choice persists
- **Color mode preference** — Light/dark/system is saved
- **Quick-switcher enabled status** — If you've turned it on/off
- **Across sessions** — Preferences stay the same when you log back in

Every preference you set via the quick-switcher is saved to your account.

The theme quick-switcher is designed for convenience—use it to keep theming controls at your fingertips!
