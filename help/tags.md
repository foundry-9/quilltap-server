# Tags

> **[Open this page in Quilltap](/aurora)**

Tags are labels you create to organize and categorize your characters, connection profiles, and image profiles throughout Quilltap. They're powerful tools for filtering, searching, and managing your content efficiently.

## What Are Tags?

Tags are custom keywords you assign to your content to help you organize and find things quickly. For example, you might create tags like "fantasy," "romance," "science-fiction," or "comedy" to categorize your characters by genre, or "daily," "weekly," "archived" to track project status.

## Where Tags Are Used

### Characters

Each character can have multiple tags assigned to help you organize your character roster:

- Tags appear in the character's **Tags tab** on their profile
- Tags help filter your characters in the sidebar
- Tagged characters can be hidden using the **quick-hide** feature
- When creating a new chat, you can filter character suggestions by tags

### Connection Profiles

Connection profiles (your AI provider settings) can have tags to help you match them with characters:

- Profiles tagged with matching character tags appear first when selecting a profile for a character
- For example, if a character has the "fast" tag, connection profiles also tagged "fast" will be prioritized
- This helps ensure characters use the right AI provider settings

### Image Profiles

Similar to connection profiles, image generation profiles can be tagged:

- Image profiles matching character tags appear first
- Helps automate the selection of the right image generation provider for each character
- Improves consistency across your creative work

## Finding Tags

### In Search

Use the global search bar to find content by tag:

- Type a tag name in the search bar to see all characters, profiles, and chats tagged with it
- Tags appear as a separate category in search results
- This is the fastest way to find all tagged content

### In Settings

The **Tags customization tab** in Settings is your central hub for managing all tags and their appearance:

- View all tags in your system
- Customize how each tag looks
- Mark tags as "quick-hide" enabled
- See tag usage statistics

## Tagging Content

### How to Add Tags

1. Navigate to the item you want to tag (a character, profile, or file)
2. Find the **Tags** section or **Tags tab**
3. Click to add a tag
4. Search for existing tags or create a new tag by typing
5. Click the tag to add it
6. Tags are saved automatically

### Creating New Tags

When you type a tag name that doesn't exist:

- Quilltap will offer to create it for you
- New tags are available to use on any content immediately
- You can customize the new tag's appearance in Settings

### Removing Tags

- Click the **X** on a tag to remove it from the content
- The tag itself remains in your system for use elsewhere
- You can reapply it to the same or different content anytime

## Tag Customization in Settings

The **Tags** tab in your Settings provides powerful customization options for each tag's appearance:

### Visual Style Options

Each tag can be customized with:

- **Color** — Choose the tag's background and text colors using the color picker
- **Emoji** — Add a visual icon to the tag (optional)
- **Emoji-only mode** — Display only the emoji without the tag name for a compact look
- **Text styling** — Make the tag text bold, italic, or strikethrough for emphasis

### Preview

As you customize a tag, you'll see a live preview of how it will appear throughout Quilltap. This helps you choose colors and styling that work well together.

### Quick-Hide Option

At the bottom of each tag's customization panel, you'll find the **Quick Hide** toggle:

- Enable this if you want this tag to be available for quick-hide
- Only tags with quick-hide enabled appear in the quick-hide menu
- See the "Quick-Hide Feature" section below for more details

### Saving Changes

All customization changes are saved automatically as you edit them. The tag's new appearance will be reflected across Quilltap immediately.

## Quick-Hide Feature

The quick-hide feature lets you temporarily hide content tagged with specific tags, helping you focus on what matters most.

### How Quick-Hide Works

1. Look for the **Quick-Hide icon** (eye icon) in the sidebar footer
2. Click it to open the quick-hide menu
3. You'll see a list of all tags marked as "quick-hide enabled"
4. Click any tag to toggle its visibility:
   - **Open eye** = tag is visible (content is shown)
   - **Closed eye** = tag is hidden (content is hidden)

### What Gets Hidden

When you hide a tag, content tagged with that tag will be hidden from:

- **Character sidebar** — Characters with hidden tags don't appear in the sidebar
- **Chat sidebar** — Chats involving only characters with hidden tags are removed from the sidebar
- **Homepage** — Favorite characters with hidden tags are not displayed
- **Quick action lists** — Characters with hidden tags are filtered from suggestions

### Why Use Quick-Hide

- **Focus on active work** — Hide archived or inactive characters while working on current projects
- **Reduce clutter** — Temporarily hide characters you're not actively using
- **Manage scope** — Hide genre-specific characters when working on a different genre
- **Clean interface** — Keep your sidebar showing only relevant content

### Quick-Hide vs. Deleting

Quick-hide is temporary and non-destructive:

- Hidden content still exists in your system
- You can unhide it anytime by toggling the tag again
- No data is lost or deleted
- Quick-hide settings are saved per user

## Managing Tags in Settings

### Tag List

In the Tags customization tab, you'll see:

- **All tags** in your system listed in order
- **Tag preview** showing how each tag appears with its customization
- **Quick-hide status** indicator for each tag
- **Edit option** to customize each tag's appearance

### Deleting Tags

The **Tag Management** section at the bottom of the Tags tab lets you delete any tag. Each tag shows its usage count across all entity types. When deleted, the tag is automatically removed from all characters, chats, profiles, and files that use it. See the [Tag Customization](tags-customization.md) page for full details.

### Organizing Your Tags

Keep your tag system organized:

- Use **consistent naming** — Decide on a naming convention (e.g., lowercase, underscores for multi-word tags)
- **Create meaningful tags** — Use tags that have clear purpose and meaning
- **Avoid tag explosion** — Don't create too many similar tags; consolidate related tags
- **Document your system** — In a note or character description, document what your tags mean

### Best Practices

1. **Genre tags** — Create tags for fiction genres (fantasy, sci-fi, romance, horror, etc.)
2. **Status tags** — Track project state (active, archived, wip, completed)
3. **Source tags** — Mark content source (original, fanfiction, imported, etc.)
4. **Availability tags** — Mark LLM profile appropriateness (fast, cheap, creative, etc.)
5. **Frequency tags** — Track usage patterns (daily, weekly, occasional, inactive)

## Using Tags for Filtering

### In Profile Selection

When setting up a character's connection profile or image generation profile:

- Profiles with matching tags appear at the top of the list
- This helps you quickly find the right profiles for each character
- Saves time and ensures consistent setup

### In Character Selection

When creating a new chat:

- Filter character suggestions by tags
- Find specific character types quickly
- Reduce the time spent searching through your full roster

### In Search

Use tags as search filters to find content faster:

- Search by tag name to see all related content
- Combine multiple tag searches for complex filtering
- Tags help you organize and discover your content

## Tips for Effective Tagging

1. **Be consistent** — Use the same tags across similar content
2. **Use multiple tags** — A character can have many tags; use them!
3. **Review regularly** — Periodically check and update tags as your projects evolve
4. **Quick-hide strategically** — Set up quick-hide tags for major groupings (e.g., "archived," "inactive")
5. **Combine with visual styles** — Use distinctive colors and emojis for quick visual identification

Tags are flexible and personal—develop a tagging system that works for your creative workflow!
