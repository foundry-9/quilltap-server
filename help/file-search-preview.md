# Searching and Previewing Files

> **[Open this page in Quilltap](/files)**

Learn how to find and preview your files in Quilltap.

## Finding Files

### Search Box

The fastest way to find files:

1. Look for **Search** field in file browser (usually at top)
2. Type file name or keywords (e.g., "character", "research")
3. Results filter in real-time
4. See matching files as you type
5. Clear search to see all files again

**Search matches:**

- File names
- File descriptions (if added)
- Folder names
- File types

### Filtering Files

**By Type:**

If available, filter view to show only:

- Images
- Documents
- Code files
- PDFs
- etc.

Look for filter buttons or dropdowns in toolbar.

**By Size:**

Filter files by file size:

- Very small (< 1 MB)
- Small (1-10 MB)
- Medium (10-100 MB)
- Large (> 100 MB)

**By Date:**

Filter by when file was created/modified:

- Today
- This week
- This month
- This year
- Older

### Sorting Files

Click column headers or sorting options:

**Sort by:**

- **Name** — Alphabetical (A to Z or Z to A)
- **Type** — Group by file type
- **Size** — Largest first or smallest first
- **Date** — Newest first or oldest first
- **Modified** — Most recently changed first

**Sort order:**

- Click once for ascending (A→Z)
- Click again for descending (Z→A)
- Arrow indicator shows current sort direction

### Using Folder Navigation

**Logical browsing:**

1. Know which folder contains files you want
2. Navigate there using breadcrumb or folder clicks
3. All files in that folder display
4. Scroll or sort to find specific file

**Example:** `/Projects/Novel-2025/Characters/` shows all character files

### Advanced Search

If advanced search available:

1. Click **Advanced Search** or **Options**
2. Set multiple criteria:
   - File name contains: [text]
   - File type is: [type]
   - Size greater than: [size]
   - Date modified: [date range]
3. Click **Search**
4. Results show matching files

## Previewing Files

### Opening File Preview

**Method 1: Click File Name**

1. In file browser, click file name
2. Preview modal opens
3. Shows file content in appropriate format
4. Can navigate to other files from preview

**Method 2: Preview Button**

1. Look for **Preview** button (eye icon)
2. Next to file in list or on hover
3. Click to open preview modal

**Method 3: Double-Click**

Some systems open preview on double-click instead of folder entry

### Understanding Preview Types

#### Image Files

**What you see:**

- Full image display (scaled to fit)
- Image dimensions (e.g., 1920x1080)
- File size
- Image format (JPG, PNG, etc.)

**Actions:**

- **Zoom In/Out** — Use controls or scroll wheel
- **Pan** — Click and drag to move around
- **Download** — Save to computer
- **Full Screen** — View in full screen mode

**Navigation:**

- Arrow buttons to browse other images in folder
- Arrow keys on keyboard (← →)
- File counter shows position (e.g., "2 of 5")

#### PDF Files

**What you see:**

- PDF document with all pages
- Page navigation controls
- Current page indicator
- Search within PDF

**Actions:**

- **Next/Previous** — Navigate pages
- **Jump to Page** — Enter page number
- **Zoom** — Increase/decrease page size
- **Search** — Find text in PDF
- **Download** — Save PDF to computer
- **Full Screen** — View full screen

**Reading Tips:**

- PDFs maintain original formatting
- Large PDFs may take moment to load
- Can select and copy text

#### Text Files

**What you see:**

- File content with syntax highlighting
- Line numbers (for code files)
- Proper formatting and indentation
- Color-coded syntax (if code)

**File Types Supported:**

- Plain text (.txt)
- Code files (JavaScript, Python, etc.)
- JSON, XML, YAML
- HTML, CSS
- Markdown files

**Actions:**

- **Copy Content** — Select and copy text
- **Download** — Save to computer
- **Full Screen** — Expand view
- **Line Number Toggle** — Show/hide line numbers

#### Markdown Files

**Special Features:**

- Full markdown rendering
- Headings, bold, italics, lists formatted
- Code blocks with syntax highlighting
- Links clickable (if pointing to local files)

**Wikilinks Support:**

- `[[filename]]` links to other files
- Click to jump to referenced file
- `[[file#heading]]` jumps to section in file

**YAML Frontmatter:**

- Metadata at top of file displayed separately
- Key-value pairs shown in structured format
- Example: `title:`, `author:`, `tags:`

**Table of Contents:**

- Heading navigation (if available)
- Jump to specific section
- Outline view of document structure

#### Code Files

**Features:**

- Syntax highlighting by language
- Line numbers showing position
- Color-coded syntax (keywords, strings, etc.)
- Proper indentation preserved

**Supported Languages:**

- JavaScript/TypeScript
- Python
- Java, C++, C#
- Ruby, Go, Rust
- PHP, Java
- HTML, CSS, SCSS
- JSON, XML, YAML
- Bash, PowerShell
- SQL
- And many more

**Actions:**

- **Copy Code** — Select and copy
- **Download** — Save to computer
- **Full Screen** — Larger view
- **Line Wrapping** — Toggle long line wrapping

#### Unsupported File Types

**When preview not available:**

1. File type can't be displayed in browser
2. Shows file information instead:
   - File name
   - File size
   - File type
   - Creation date
   - Last modified date

3. Download button provided

**Example:** Binary files, executables, compressed archives

### Navigating Between Files

**While in Preview:**

1. **Previous/Next Buttons** — Top of preview modal
   - ← Previous file
   - → Next file
2. **Arrow Keys** — Press ← or → on keyboard
3. **File Counter** — Shows "2 of 5" (current of total)
4. **Loop back** — At end, goes back to first file

**Browse Order:**

- Files shown in same order as file browser
- Current sort order determines browse sequence
- Sorted by name, date, size, etc. determines order

### Preview Controls

**Common Controls:**

| Control | Action |
|---------|--------|
| 🔍 | Zoom in |
| 🔍- | Zoom out |
| ⊡ | Fit to screen |
| ⛶ | Full screen mode |
| ↓ | Download file |
| ← → | Previous/next file |
| ✕ | Close preview |

**Keyboard Shortcuts:**

| Key | Action |
|-----|--------|
| Esc | Close preview |
| ← | Previous file |
| → | Next file |
| + | Zoom in |
| - | Zoom out |
| Ctrl+C | Copy content (text files) |
| Ctrl+F | Find in PDF/text |
| Space | Page down (PDFs) |

### Full Screen Preview

To view file in full screen:

1. Click **Full Screen** button (⛶ icon)
2. Preview expands to fill entire screen
3. Minimizes UI elements
4. Press Esc to exit full screen

**Good for:**

- Reading long documents
- Viewing images in detail
- Focusing on content
- Presentations

## File Information and Metadata

### Viewing File Details

**In List View:**

Columns typically show:

- **Name** — File name
- **Type** — File type (PDF, JPG, etc.)
- **Size** — File size (KB, MB)
- **Date** — Created or modified date
- **Associations** — What uses this file

**On Hover (Grid View):**

Hovering over file shows:

- File name
- File size
- File type
- Date
- Quick action buttons

### File Properties

**Full information access:**

1. Right-click file
2. Select **Properties** or **Details**
3. Shows:
   - File name and path
   - File size (bytes, KB, MB)
   - File type (MIME type)
   - Created date and time
   - Modified date and time
   - Dimensions (if image)
   - Associations (characters, chats)

### File Descriptions

**Adding descriptions:**

1. Find file in browser
2. Click **Edit** or **Add Description**
3. Type description of file contents
4. Save
5. Description visible on hover or in details

**When to add:**

- Complex files that need explanation
- Files with unclear names
- Important files you want to remember
- Files for sharing with others

## Quick File Access

### Recent Files

If available, quick access section shows:

- Recently uploaded files
- Recently modified files
- Recently viewed files

Click to quickly jump to file.

### Frequently Used

Favorites or pinned files section:

1. Click pin or star icon on file
2. File appears in quick access area
3. Easily find most-used files

### Breadcrumb Shortcuts

Path bar shows quick navigation:

- Click any folder name to jump there
- Click **Home** or root (/) to go to top
- Shows full current path

## File Search Tips

### Effective Searching

**Be specific:**

- ✓ "character-alice-profile" — Good
- ✗ "files" — Too vague

**Use keywords:**

- Search for main topic
- Results show similar files
- Helps discover related content

**Use wildcards (if supported):**

- `*.pdf` — All PDF files
- `character-*` — Files starting with "character-"
- `*research*` — Files containing "research"

### Finding Similar Files

1. Search for main topic
2. See all related files
3. Spot duplicates or unused versions
4. Delete or consolidate as needed

### Searching by Type

1. Use filter for file type
2. See all images, or all PDFs, etc.
3. Easier than searching by name

## Troubleshooting Search and Preview

### Search not finding file

**Causes:**

- File name misspelled in search
- File in different location than expected
- File deleted

**Solutions:**

- Check spelling
- Browse folders to find location
- Verify file still exists

### Preview not working

**Causes:**

- File type not supported
- File corrupted
- Very large file
- Browser issue

**Solutions:**

- Try downloading instead
- Check file integrity (re-upload if possible)
- Try smaller file first
- Use different browser

### Preview very slow

**Causes:**

- Large file
- Slow internet
- Server busy
- PDF with many pages

**Solutions:**

- Wait for preview to load
- Check internet connection
- Try accessing later
- Download file instead of preview

### Can't see file properties

**Causes:**

- File permissions issue
- API error
- File deleted

**Solutions:**

- Verify file still visible in browser
- Try refreshing page
- Check file still exists
