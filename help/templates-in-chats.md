# Using Templates in Chats

> **[Open this page in Quilltap](/salon)**

This guide explains how to select and use roleplay templates in your individual chats. Each chat can have its own template, or it can use the default template from your settings.

## Understanding Templates in Chats

### What the Template Controls

When a template is active in a chat, it:

- **Formats AI responses** — Shapes how the character's responses appear
- **Provides instructions** — Guides the AI on formatting conventions
- **Applies system prompts** — Prepends formatting rules to character instructions
- **Affects all participants** — All characters in the chat follow the same format

### Template Scope

- **Per-chat** — Each chat can have its own template
- **Can override default** — Any chat can use a different template than the default
- **Independent** — Changing one chat's template doesn't affect others
- **Can be changed anytime** — Switch templates at any point in a conversation

## Accessing Chat Settings

### Opening Chat Settings

While in a chat:

1. Look for the **settings icon** (gear icon) at the top right
2. Click it to open **Chat Settings**
3. The settings modal appears
4. Scroll to the **Roleplay Template** section

### What Settings Include

The Chat Settings modal contains:

- **Roleplay Template** — Template selection dropdown
- **Participants** — List of characters in the chat
- **Other settings** — Avatar display, chat history, etc.

## Selecting a Template

### The Template Dropdown

The template selection shows:

- **Current template** — What's currently selected
- **"None"** — Option to use no template
- **All available templates** — Your custom and built-in templates
- **"Built-in" label** — Indicates which are built-in vs. custom

### Choosing a Template

1. Click the template dropdown
2. Browse the available options:
   - **None** — No formatting template
   - **Built-in templates** — Pre-installed formats
   - **Your custom templates** — Templates you've created
3. Click the template you want to use
4. It's selected immediately

### What "None" Means

Selecting "None" template:

- **No formatting** — The AI won't follow specific formatting rules
- **Default behavior** — The AI uses its natural response style
- **Still useful** — Some chats don't need strict formatting
- **Can change later** — Select a template anytime

## Saving Template Changes

### Automatic Saving

When you select a template:

1. Click the template in the dropdown
2. A "Saving..." message appears briefly
3. Your change is saved to the chat
4. The modal stays open for other adjustments

### Confirming the Change

After selecting a template:

- **Saving indicator** — "Saving..." shows while it's being saved
- **Success** — Message confirms the change was saved
- **Applied immediately** — New responses use the new template
- **Can select again** — Choose a different template anytime

### If Saving Fails

If the change doesn't save:

- An error message appears
- The dropdown reverts to the previous template
- Try selecting again
- Check your internet connection
- Close and reopen Chat Settings if stuck

## Impact of Template Changes

### When You Switch Templates

When you change a chat's template:

- **Previous messages** — Keep their current formatting
- **New messages** — Use the new template's formatting
- **Formatting changes** — New AI responses follow the new rules
- **User messages** — Aren't affected by template change

### Timeline

```
Chat history:
[Message 1 - Old template format]
[Message 2 - Old template format]
>> You switch templates <<
[Message 3 - New template format]
[Message 4 - New template format]
```

### Why Mix Formatting?

You might see mixed formatting when you:

- **Tested templates** — Tried different templates, kept good results
- **Evolved your style** — Your template requirements changed
- **Experimented** — Switched templates to see what works
- **Recovered a chat** — Changed templates mid-conversation

## Using Templates Effectively

### Choosing the Right Template

When selecting a template for a chat:

1. **Consider the tone** — Is the chat dramatic, comedic, serious?
2. **Match the setting** — Fantasy, sci-fi, realistic, etc.
3. **Think about pacing** — Fast-paced chats vs. detailed ones
4. **Review character needs** — Does the character need specific formatting?
5. **Test if unsure** — Try the template and see if it works

### Template for Different Chat Types

- **Story-heavy chats** — Use templates with strong narration
- **Dialogue-focused** — Use templates emphasizing dialogue
- **Character development** — Use templates showing thoughts and feelings
- **Action scenes** — Use templates for clear action descriptions
- **Mixed content** — Use balanced templates

### Changing Templates Mid-Conversation

You can switch templates during a chat:

1. Open Chat Settings
2. Select a new template
3. Click Save
4. The next AI response uses the new format
5. Useful if you realize a template isn't working

### Reverting to Default

If you set a template and want to go back to default:

1. Open Chat Settings
2. Find the Roleplay Template dropdown
3. Note which template is the default (from Settings)
4. Select it in the dropdown
5. Your chat returns to the default template

## Template Information

### Viewing Template Details

To learn more about the selected template:

1. Open Chat Settings
2. Find the template dropdown
3. Hover over a template name for basic info
4. Go to **Settings** → **Roleplay Templates** for full details
5. Click **Preview** to see the complete system prompt

### Understanding System Prompts

The system prompt contains:

- **Formatting rules** — How to structure different elements
- **Examples** — What correctly formatted text looks like
- **Instructions** — How the AI should format responses
- **Conventions** — Special rules or styles

### Templates with Special Features

Some templates include:

- **Annotation buttons** — Quick formatting tools
- **Rendering patterns** — How different text appears
- **Dialogue detection** — Automatic dialogue identification
- **Multiple formatting options** — Flexibility in how you write

## Common Scenarios

### Scenario 1: Starting a New Chat

1. Create a new chat
2. Open Chat Settings
3. The default template is already selected
4. If you want a different template, select it now
5. Close Settings and start chatting

### Scenario 2: Switching Styles Mid-Story

1. You're in a chat that was dramatic
2. The story shifts to action/adventure
3. Open Chat Settings
4. Switch to an action-focused template
5. Your new responses are formatted for action

### Scenario 3: Fixing Formatting Issues

1. Responses aren't formatted well
2. Open Chat Settings
3. Try a different template that might work better
4. Send a test message to see if it helps
5. Keep the template if it works, try another if not

### Scenario 4: Matching Multiple Characters

1. Your chat has multiple characters
2. You want consistent formatting across all of them
3. Select one template that works for everyone
4. All characters will follow that template's rules
5. Switch templates if needed to find one that works for all

## Best Practices

### Setting Up Templates

1. **Test first** — Try templates in experimental chats
2. **Find your favorite** — Discover what works for your style
3. **Set a good default** — Use your favorite as the default
4. **Document your choice** — Remember why you chose a template
5. **Review occasionally** — Check if templates still match your needs

### Using Templates Effectively

1. **Match the mood** — Template should fit the chat's tone
2. **Consistency** — Use the same template for similar chats
3. **Flexibility** — Don't be afraid to switch templates
4. **Test changes** — Verify a new template works before relying on it
5. **Keep notes** — Remember which templates work for what

### When Templates Don't Work

1. **Try a different template** — Your first choice might not be ideal
2. **Combine approaches** — Use one template for some chats, another for others
3. **Create custom** — If no template works, create your own
4. **Adjust character prompts** — Sometimes the character instruction matters more
5. **Accept mixed styles** — Not every chat needs perfect formatting

## Troubleshooting

### Template Not Showing in List

**Problem:** A template I created doesn't appear in the dropdown

**Solutions:**

- Refresh Chat Settings
- Go to the **Templates** tab in Settings (`/settings?tab=templates`) → Roleplay Templates to verify it was created
- Make sure you're looking in the right section
- Try closing and reopening Chat Settings
- Refresh the entire page

### Template Not Applying

**Problem:** Responses don't follow the template format

**Solutions:**

- Verify the template is selected (check Chat Settings)
- Try with a simpler template first to verify it works
- Send a new message to test the template
- Check the character's system prompt for conflicting instructions
- Try switching to a different template

### Responses Look Same as Before

**Problem:** Template change seems to have no effect

**Solutions:**

- Templates affect *new* messages, not existing ones
- Send a new test message to see the template applied
- Check that the template is actually different from the previous one
- Verify the template isn't "None" (which means no formatting)
- Try switching back to the original template, then to a new one

### Can't Change Template

**Problem:** The template dropdown is disabled or won't work

**Solutions:**

- Check that Chat Settings modal is fully loaded
- Refresh the page
- Verify you're logged in
- Check your internet connection
- Try closing and reopening Chat Settings

### Template Reverts

**Problem:** My template selection keeps reverting

**Solutions:**

- Wait for "Saving..." to complete before closing settings
- Check your internet connection
- Try selecting the template again
- Refresh the page to see what was actually saved
- Look for error messages

## Tips for Success

1. **Start with defaults** — Use the default template as a baseline
2. **Test templates early** — Experiment with different ones at the start
3. **Switch if needed** — Don't stick with a template that isn't working
4. **Document your choice** — Remember which template you chose and why
5. **Backup good settings** — Save chat settings that work well
6. **Review periodically** — Update templates as your preferences evolve
7. **Ask for feedback** — Share templates with others to get suggestions
8. **Keep learning** — Read template documentation to use them better

The template system gives you fine-grained control over how your chats look and feel—use it to create the perfect environment for your creative work!
