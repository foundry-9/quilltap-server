# Organizing Characters

> **[Open this page in Quilltap](/aurora)**

This guide covers how to organize, tag, filter, and manage your character collection.

## Understanding Character Organization

As your character collection grows, organization becomes important. Quilltap provides several tools to keep characters organized and easy to find:

- **Tags** — Label characters with customizable tags
- **Favorites** — Mark frequently-used characters
- **Quick Hide** — Temporarily hide characters from view
- **Filtering** — Show/hide characters by tags and status
- **Sorting** — Sort by name, created date, modified date
- **Search** — Find characters by name and properties

## Using Tags

Tags are the primary way to organize characters.

### Understanding Tags

Tags are customizable labels you can apply to characters:

**Examples:**

- Story/Campaign: "Main Characters", "Companions", "NPCs", "Antagonists"
- Genre: "Fantasy", "Sci-Fi", "Modern", "Historical"
- Status: "Active", "Archived", "Testing", "Draft"
- Type: "Human", "AI", "Animal", "Supernatural"
- Relationship: "Allies", "Enemies", "Mentors", "Love Interests"

### Adding Tags When Creating

1. Create or edit character
2. Scroll to **Tags** section
3. Click **Add Tag** or tag input field
4. Type existing tag name or create new
5. Select or confirm tag
6. Save character

**Tip:** Start typing to search existing tags or create new ones.

### Adding Tags to Existing Character

**Method 1: From Edit Screen**

1. Edit character
2. Scroll to Tags section
3. Click **Add Tag**
4. Select or create tag
5. Save

**Method 2: From Character List**

1. Hover over character
2. See tag section
3. Click **+ Add Tag**
4. Select or create tag
5. Auto-saves

**Method 3: Bulk Add**

1. Select multiple characters (if view supports multi-select)
2. Click **Bulk Actions** or right-click menu
3. Select **Add Tags**
4. Choose tags to add
5. Confirm

### Removing Tags

**From Edit Screen:**

1. Edit character
2. Locate tag in Tags section
3. Click **X** next to tag
4. Save

**From Character List:**

1. Hover over character
2. Click **X** on tag
3. Tag immediately removed

### Organizing with Tags: Common Strategies

**Strategy 1: Campaign Organization**

- Tag by story: "Campaign: Lost City", "Campaign: Dragon War"
- Helps track characters across story arcs
- Easy to show/hide all campaign characters at once

**Strategy 2: Character Type**

- Tag by role: "NPCs", "Main Characters", "Temporary"
- Makes it easy to see all NPCs in one view
- Separate main story from side characters

**Strategy 3: Status Tracking**

- Tag status: "Active", "Archived", "Draft", "Testing"
- Helps manage character lifecycle
- Archive completed characters without deleting

**Strategy 4: Genre/Setting**

- Tag by genre: "Fantasy", "Sci-Fi", "Modern", "Horror"
- For writers with multiple story worlds
- Easy to see characters from specific setting

**Strategy 5: Hybrid Organization**

```
Example character tags:
- "Campaign: Lost City"
- "Main Characters"
- "Fantasy"
- "Active"
- "NPCs: Vendors"
```

### Tag Best Practices

| Do | Don't |
|----|-------|
| Use consistent tag names | Use "fantasy" and "FANTASY" separately |
| Keep tag count reasonable (3-10 per character) | Tag every possible attribute |
| Use descriptive tags | Use vague abbreviations like "X1" |
| Create hierarchy with prefixes ("Campaign: ", "Type: ") | Create flat list of 100+ tags |
| Review tags periodically | Never update tags |

### Creating Tag System

**Step 1: Plan your tags**

- What stories/campaigns are you writing?
- What character types do you need?
- What statuses matter? (Active, Draft, etc.)
- What genres do you write in?

**Step 2: Create tag categories**

```
Campaign tags:
- Campaign: [Story Name]

Character type tags:
- Main Characters
- Supporting Characters
- NPCs
- Antagonists
- Love Interests

Status tags:
- Active
- Archived
- Draft
- Testing

Genre tags:
- Fantasy
- Sci-Fi
- Modern
- Horror
```

**Step 3: Apply consistently**

- When creating character, apply relevant tags
- Keep tag set consistent across characters
- Review and update periodically

**Step 4: Use for filtering (see below)**

- Filter by tag when looking for specific characters
- Create smart lists of characters by tag

## Filtering and Searching

### Basic Filtering

The character list has filter controls:

**Filter by Tag:**

1. Go to **Characters** page
2. See **Filters** or **Tags** section
3. Click tag to filter
4. List shows only characters with that tag
5. Click tag again to remove filter
6. Click **Clear Filters** to show all

**Filter by Status:**

1. Some views show status filters
2. Filter for: Active, Archived, Draft, etc.
3. See only characters with selected status

### Advanced Filtering

**Multi-tag filtering:**

1. Click first tag to filter
2. Click additional tags to add to filter
3. See characters matching ANY selected tag
4. Or characters matching ALL tags (depends on view)

**Toggling filters:**

- Click tag to add filter
- Click filled tag to remove filter
- Click **Clear All** to reset

### Searching Characters

**Search by name:**

1. Click **Search** box in character list
2. Type character name (partial match works)
3. List filters to matching characters
4. Results show as you type

**Search results show:**

- Character name
- Tags
- Last modified date
- Avatar/image

**Tips:**

- Search is case-insensitive
- Works with partial names ("ali" finds "Alice")
- Returns results instantly

### Sorting Characters

Different views offer different sorting options:

**Common sort options:**

- **Name (A→Z)** — Alphabetical by character name
- **Name (Z→A)** — Reverse alphabetical
- **Recently Modified** — Characters changed most recently first
- **Created Date** — Newest characters first
- **Favorites** — Favorite characters at top

**How to sort:**

1. Look for **Sort** menu or option
2. Select sort method
3. Character list reorganizes
4. Sort selection persists until changed

## Using Favorites

Favorite characters appear at top and get special treatment.

### Marking as Favorite

**Method 1: From Character List**

1. Hover over character
2. Click **Star** icon
3. Character marked as favorite
4. Star fills in solid
5. Character moves to favorites section at top

**Method 2: From Character View/Edit**

1. Open character view or edit
2. Click **Star** icon at top
3. Character marked favorite
4. Star fills in

### Removing from Favorites

Click star again to unfavorite.

**Visual indicator:** Filled star = favorite, Empty star = not favorite

### Uses for Favorites

- **Frequently used characters** — Favorite main characters
- **Active characters** — Favorite characters in current campaign
- **Recently created** — Favorite new characters to easily find
- **Testing** — Favorite characters you're working on

**Tip:** Keep favorites minimal (5-10) so they're actually useful.

## Quick Hide

Quick Hide temporarily hides characters from normal view.

### Hiding Characters

**Method 1: From Character List**

1. Hover over character
2. Right-click or click menu
3. Select **Quick Hide**
4. Character disappears from list

**Method 2: From Character View**

1. Open character
2. Click menu (⋮ or ⋯)
3. Select **Quick Hide**
4. Redirected to character list
5. Character no longer visible

### Viewing Hidden Characters

**Show hidden characters:**

1. Character list has **Show Hidden** toggle or filter
2. Toggle **ON** to see hidden characters
3. Hidden characters appear with hide indicator
4. Can still interact with them

### Unhiding Characters

**To restore hidden character:**

1. Toggle **Show Hidden** on
2. Find hidden character
3. Right-click or menu
4. Select **Unhide** or **Show**
5. Character returns to normal view

### Uses for Quick Hide

- **Spoiler characters** — Hide to avoid accidental spoilers
- **Future characters** — Hide characters planned for later
- **Retired characters** — Hide completed character arcs
- **Testing characters** — Hide test characters from main view
- **Organization** — Hide characters not currently in use

**Difference from Delete:** Quick Hide keeps character but hides it. Delete removes character permanently.

## Character Collections and Smart Filters

### Creating Collections (If Available)

Some views support creating collections:

1. Click **+ New Collection**
2. Give collection a name
3. Drag characters into collection
4. Or use rules-based collection (auto-include by tag)
5. Collection appears as saved view

### Saved Filters

1. Set up filter (by tags, status, etc.)
2. Click **Save Filter**
3. Give filter a name
4. Filter saved for quick access
5. Can reuse saved filter anytime

## Bulk Operations

If your view supports multi-select:

### Bulk Tag

1. Select multiple characters (checkbox or Cmd+Click)
2. Right-click or click **Bulk Actions**
3. Select **Add Tags**
4. Choose tags
5. Confirm
6. All selected characters tagged

### Bulk Hide

1. Select characters
2. Click **Bulk Actions** > **Quick Hide**
3. All selected characters hidden

### Bulk Favorite

1. Select characters
2. Click **Bulk Actions** > **Add to Favorites**
3. All selected characters favorited

### Bulk Delete (Use Caution!)

1. Select characters
2. Click **Bulk Actions** > **Delete**
3. Confirmation appears
4. Cannot undo
5. All selected characters deleted

## Workspace Organization Workflows

### Workflow 1: Campaign Organization

You're running a fantasy RPG campaign:

**Setup:**

1. Create tags: "Campaign: Lost City", "Main Characters", "NPCs", "Antagonists"
2. Create characters with appropriate tags
3. Create saved filter "Campaign: Lost City"

**Maintain:**

- When creating new character, tag appropriately
- Use saved filter to see only campaign characters
- Archive campaign characters when campaign ends

### Workflow 2: Multi-Genre Writer

You write fantasy, sci-fi, and modern stories:

**Setup:**

1. Create tags: "Fantasy", "Sci-Fi", "Modern"
2. Add type tags: "Main Characters", "Supporting"
3. Create saved filters for each genre

**Maintain:**

- Tag all characters by genre
- Use genre filters to see story-specific characters
- Keep genre-organized view

### Workflow 3: NPC Library

You run ongoing games with many NPCs:

**Setup:**

1. Create tags: "NPCs", "Recurring NPCs", "One-Shots"
2. Create location tags: "NPCs: Tavern", "NPCs: Market"
3. Create saved filter "All NPCs"

**Maintain:**

- Tag all NPCs with location/type
- Quick hide used NPCs that won't return
- Archive one-shots after session

### Workflow 4: Character Development

You're building character library for general use:

**Setup:**

1. Create status tags: "Draft", "Work in Progress", "Complete", "Published"
2. Create archetype tags: "Wizard", "Rogue", "Warrior", "Cleric"
3. Create saved filter "Complete Characters"

**Maintain:**

- Tag characters by development status
- Update status as you work
- Archive complete characters
- Export completed characters

## Search and Discovery

### Finding Forgotten Characters

If you can't remember character name:

1. Open Characters page
2. Sort by **Recently Modified**
3. Scroll through recent characters
4. Or filter by tag if you remember one

### Rediscovering Archived Characters

1. Check **Show Hidden** toggle
2. Or check **Show Archived** toggle
3. Look through hidden/archived section
4. Click **Unhide** to restore

### Using Search Effectively

**Search scenarios:**

- "alice" → finds Alice, Alice's Friend, Alicia
- "detective" → finds any character with Detective in name or title
- Partial matches work
