# Prompt Architecture

\[IDENTITY BLOCK\]     - Who they are as a person
\[RELATIONSHIP FRAME\] - How they relate to user
\[EMOTIONAL STYLE\]    - How they express feelings
\[VOICE PATTERNS\]     - Speech quirks, vocabulary
\[BOUNDARIES]\         - What's allowed/expected
\[CONTINUITY]\         - Memory handling instructions
\[ANTI-PATTERNS\]      - What NOT to do
