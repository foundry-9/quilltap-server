# Deleting Your Data

> **[Open this page in Quilltap](/settings?tab=system&section=delete-all-data)**

The Delete All Data tool permanently removes your entire Quilltap account and all associated data.

## ⚠️ Warning - This Action Is Permanent

**Deleting data:**

- Cannot be undone
- Removes all your information from the system
- Data is permanently deleted (not recoverable)
- Should only be done if you're certain

**Before proceeding, create a backup** if you might want this data in the future.

## What Gets Deleted?

A complete data deletion removes:

**Chat & Message Data**

- All chats and conversations
- All messages and replies
- Chat settings and configurations

**Characters**

- All character profiles
- Character configurations
- Character images
- All character-specific data

**Memories**

- All extracted memories
- Memory metadata
- Memory relationships
- Memory analysis data

**Files & Media**

- All uploaded files
- All images
- All media attachments
- File organization and tags

**Settings & Profiles**

- Connection profiles
- Image generation profiles
- API keys and credentials
- Embedding profiles
- User preferences

**Templates & Prompts**

- Roleplay templates
- System prompts
- Saved prompts
- Template configurations

**Organization**

- Tags and categories
- Projects
- Collections
- Custom organization

**Account Data**

- Account information
- Login history
- API keys
- Sync data
- All account settings

**Backups**

- Stored backups (cloud and local)
- Export files
- Archives
- Recovery data

## Deleting Your Data

**Important:** Take time to consider if you really want to delete everything.

### Step-by-Step

1. **Go to the **AI Providers** tab in Settings** (`/settings?tab=providers`)

2. **Find the Delete All Data card**

3. **Click "Delete All Data"** button

4. **Preview what will be deleted**
   - Dialog shows summary of what will be removed
   - Lists counts of characters, chats, files, etc.
   - Shows storage space that will be freed
   - Review carefully

5. **Understand the consequences**
   - Read the warning message
   - Understand it cannot be undone
   - Confirm you're ready

6. **Confirm deletion**
   - You're asked to type a confirmation phrase
   - Typing confirms you understand consequences
   - Button remains disabled until you type confirmation
   - This prevents accidental deletion

7. **Click "Permanently Delete"**
   - Final confirmation
   - Deletion begins
   - May take several minutes

8. **Deletion in progress**
   - System deletes all data
   - Shows progress
   - Don't close browser tab during deletion

9. **Deletion complete**
   - Account is now empty
   - All data permanently removed
   - You're logged out
   - Can optionally delete account entirely

## Before Deleting: Checklist

**Create backups:**

- ☐ Full backup created and saved
- ☐ Important data exported
- ☐ Files downloaded locally
- ☐ Backups stored safely

**Say goodbye to data:**

- ☐ Screenshot important information
- ☐ Export chats you want to keep
- ☐ Download character definitions
- ☐ Note important settings

**Inform others:**

- ☐ Notify team members if shared instance
- ☐ Transfer responsibilities
- ☐ Archive shared data
- ☐ Update any documentation

**Final preparation:**

- ☐ Ensure you have access to backups
- ☐ Double-check file downloads
- ☐ Document any setup that took time
- ☐ Take final screenshots if needed

## Why You Might Delete Data

**Normal reasons:**

- You're done with this Quilltap instance
- Starting fresh with new instance
- Cleaning up for security
- Uninstalling Quilltap completely
- Privacy/data minimization

**Not recommended for:**

- Clearing temporary data (just delete individual items instead)
- Removing specific chats (delete them individually instead)
- Removing characters (delete them individually instead)
- Making space (use individual deletion instead)

## After Deleting

**What happens after deletion:**

- Your account still exists but is empty
- You can log back in to an empty instance
- You can delete the account entirely if desired
- Or start fresh with new characters/chats

**Recovering after deletion:**

- If you have a backup, you can restore it
- Otherwise, data is unrecoverable
- Exports can be re-imported to restore specific data
- Contact support if deletion was accidental

## Accidental Deletion Recovery

**If you accidentally delete your data:**

1. **Contact support immediately**
   - Provide account details
   - Explain what happened
   - Request recovery if possible

2. **Check for backups**
   - Do you have a backup file?
   - Can backup be restored?
   - Is backup complete?

3. **Restore from backup**
   - If backup exists, data may be recoverable
   - Use Backup & Restore tool
   - May take time to recover

4. **Re-import from exports**
   - If you have export files, re-import them
   - May lose data added since export
   - Better than complete loss

**Important:** Data recovery is not guaranteed. Always backup first.

## Alternatives to Full Deletion

**If you don't want everything:**

**Delete individual items:**

- Delete specific chats one at a time
- Delete specific characters
- Remove specific memories
- Clear specific files

**Clear data by category:**

- Delete all chats but keep characters
- Delete memories but keep chats
- Clear files but keep everything else
- Remove specific types of data

**Archive instead:**

- Export data to backup file
- Don't delete, just store offline
- Can re-import later if needed
- Keeps data safe but hidden

**Start fresh gradually:**

- Create new characters
- Start new chats
- Ignore old data
- Gradually transition without deletion

## Billing & Account Deletion

**After data deletion:**

- Active subscription continues
- Backups may still incur storage costs
- You can cancel subscription
- Contact billing if questions

**To delete account entirely:**

- Contact support
- Provide account details
- Request account deletion
- Usually permanent
- May have grace period

## Privacy & Data Deletion

**GDPR & Data Rights:**

- You have right to data deletion
- Deletion is permanent
- Complies with privacy regulations
- Contact support for compliance documentation

**Data retention:**

- We can't recover deleted data
- Logs may be retained briefly
- Backups are deleted if not manually kept
- Some audit trails may be kept for compliance
