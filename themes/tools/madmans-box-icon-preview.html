Everything is rendered as a CSS mask tinted by background-color — exactly how the app paints icons. Squint at the 16 px column: if it does not read as its verb, it fails rule 7.

Chrome blocks CSS masks on file:// pages, so the glyphs may render as empty boxes. Serve the repo instead: python3 -m http.server -d /path/to/quilltap-server 8123 then open http://localhost:8123/themes/tools/madmans-box-icon-preview.html (Safari usually renders file:// fine).
